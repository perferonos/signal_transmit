1) Скрипт make_signal_and_noise.m преобразует двоичный код из файла input.txt
в сигнал и накладывает шум. Логическая 1 = 5 В, логический 0 = 0.1 В.
Получившиеся значения сигнала записываются в файл output_signal.txt.
2) Скрипт filter_and_output накладывает фильтр на значения сигнала из файла
output_signal.txt и преобразует обратно в двоичный код. 
Получившиеся значения записываются в файл output.txt.
